Case P: Extent, Stage, Grade

⚠️ หากไม่พบ True pocket ≥ 4 mm เมื่อ charting ให้กลับไปพิจารณา Case H/G (ยกเว้นเข้า Condition Interdental CAL ≥ 2 mm หรือ B/Li CAL ≥ 3 mm with PD ≥ 4 mm และไม่เข้าข้อยกเว้น)

Extent

Stage — พิจารณาซี่ที่มี GREATEST LOSS

หรือพิจารณาจาก RBL: Coronal⅓ → Stage I/II, Middle/Apical⅓ → Stage III/IV

Complexity-shifter (ตรง ≥ 1 ข้อ → Stage III/IV)

Rehabilitation-shifter (ตรง ≥ 1 ข้อ → Stage IV)

Grade — พิจารณาซี่ที่มี GREATEST LOSS

No loss = A, <2 mm = B, ≥2 mm = C
Bone loss/Age: <0.25 = A, 0.25-1 = B, >1 = C

AAP 1999 — Classification Reference

Gingivitis

• Dental-plaque induced = Perio
• Non-dental-plaque induced = ไม่ใช่ Perio (Oral diag เช่น Leukemia, Herpes)

Chronic periodontitis

เกี่ยวกับ Local factor (Chronic plaque/calculus) — Extent + Severity (ใช้เครื่องคำนวณด้านบน)

Aggressive periodontitis

• Localized: 1st Molar + Incisor ไม่พบที่อื่น หรือ ≤ 2 ซี่
• Generalized: 1st Molar + Incisor พบที่อื่น > 3 ซี่

Necrotizing periodontal disease

• NUG: รอยแผลจำกัดที่เหงือก
• NUP: มี Attachment loss, Bone loss
• NUS: รอยแผลที่ Vestibule, Palate

Abscess of periodontium

Periodontitis associated with Endodontic lesion

• Periodontal-Endodontic lesions
• Endodontic-Periodontal lesion
• Combined lesion
